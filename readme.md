Project Euler Solutions
===

My solutions for [Project Euler](https://projecteuler.net/). ([in Japanese](http://odz.sakura.ne.jp/projecteuler/))

Languages
---

### Python

- runtime Python 3.x
- create new solution script

  ```bash
  NO=1 make new-python
  ```

### Go

- runtime Go 1.17
- create new solution script

  ```bash
  NO=1 make new-go
  ```
  
### Rust

- runtime Rust 1.56.1 or higher
- create new solution script

  ```bash
  NO=1 make new-rust
  ```
