Project Euler Solutions
===

My solutions for [Project Euler](https://projecteuler.net/). ([in Japanese](http://odz.sakura.ne.jp/projecteuler/))

# Languages

### Python

- runtime Python 3.x
- create new solution script

  ```bash
  NO=1 make new-python
  ```

### Go

- runtime Go 1.19
- create new solution script

  ```bash
  NO=1 make new-go
  ```
  
### Rust

- runtime Rust 1.63.0 or higher
- create new solution script

  ```bash
  NO=1 make new-rust
  ```
  
### Java

- runtime Java 21
- create new solution script

  ```bash
  NO=1 make new-java
  ```
